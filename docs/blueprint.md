## Core Features:

- Real-time Bus Tracking: Display real-time locations of public buses on a map with route overlays and arrival times.
- Smart Waste Bin Monitoring: Monitor the fill levels of smart waste bins and display their status (empty, medium, full) on a map.
- EV Charger Availability: Provide real-time availability and location of electric vehicle charging stations.
- Composting Reward Points: Automatically calculate and award points based on the weight of organic waste disposed of.
- Reward Wallet: Allow users to view their accumulated reward points, disposal history, and redeem points for discounts.
- Sustainable Route Planning: AI tool suggesting a combination of sustainable transportation options. Example: walking directions to the nearest bus stop that takes you to a waste disposal center or charging location.
- User Profile Management: Enable users to manage their profile information, including name, NIF, usage history, and settings.

## Style Guidelines:

- Primary color: Blue (#4A8FE7) to represent trust, technology, and the city's branding.
- Background color: Light gray (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Green (#62C769) to emphasize sustainability and environmental aspects.
- Body and headline font: 'PT Sans', a humanist sans-serif to create a balance between modern style and ease of reading, important for elderly users.
- Use friendly, rounded icons that clearly represent each smart city service.
- Web layout with a 1440px width. Left persistent sidebar for navigation. Use of rounded cards with soft shadows to display information.
- Subtle animations for map markers and data updates to enhance user experience.
